Lenguajes de programación

Html

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web HTML se considera el lenguaje web más importante siendo su invención crucial en la aparición, desarrollo y expansión de la World Wide Web (WWW) Es el estándar que se ha impuesto en la visualización de páginas web y es el que todos los navegadores actuales han adoptado.

Css

CSS (Hojas de Estilo en Cascada) es el código que usas para dar estilo a tu página web. CSS es el lenguaje que describe cómo se presenta un documento HTML. Cada usuario define las reglas que especifican los grupos de estilos que van a aplicarse a elementos particulares o grupos de elementos de la página web.

JavaScript

JavaScript es un lenguaje de programación que se utiliza principalmente para crear páginas web interactivas. Es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. JavaScript se utiliza para crear efectos visuales en la página web, validar formularios, crear animaciones y mucho más.
